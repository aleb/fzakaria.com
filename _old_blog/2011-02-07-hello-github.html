Public Repositories, A New Frontier

So today marked the day, that I had started my first repository on GitHub! I got an e-mail from someone using my HypeMachine Chrome Extension saying that he'd like to fix the bug that people are currently experiencing where navigating stops song playback. As I have limited experience in Javascript, this was wonderful news.

Long story short, the HypeMachine extension is now on GitHub! You can clone and update it and contribute to the music hive-mind. I'm eagerly waiting the release of an API on Chrome for interacting with the download manager. Once that is released, work will begin on batch downloads! =)
